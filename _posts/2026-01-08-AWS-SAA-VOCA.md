- Boot : To start up a computer or server so it begins working.

- Booting up : starting the machine.

- Hood : How something works inside.

- Encrypted : To protect data by running it into a secret code so that only people (or systems) with the correct key can used it.

- Initialize : To set something up for the first time so it's ready to be used.

- Persist data : To save data in a way that it doesen't get lost, even after the system is turned off or restarted.

- Termination : To permanently shut down and delete a resource.

- Mounted : Attaching storage (like a disk or volume) to a system so the system can use it like a folder.

- Bounding : Connected or linked to something so that they work together.

- Latency : The delay or waiting time between request and response.

- Detached : Disconnected or removed from something it was connected to.

- Provisioned : To set up, allocate or prepare resources so they're ready to use.

- Capacity : The maximum amount something can hold, handle, or support.

- Attribute : A specific piece of information or property about something.

- Preserve : To keep something safe, unchanged, and not lost or destroyed.

- Retain : To keep something not delete it, and hold onto it for future use.

- Recover : To get something back after it was lost, deleted, broken, or stopped working.

- Configuration : The settings or arragement of a system, service, or software that controls how it behave.
